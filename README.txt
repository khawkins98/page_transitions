
Module: Page Transitions 
Author: Ken Hawkins <http://AllAboutKen.com>


Description
===========
Uses hardware acceleratable CSS transforms to creat page transitions. Good for mobile effects. Applies to the the .main-container

Requirements
============

* Mobile devices that support CSS3. Shouldn't adversly affect non-supporting browswers.


Installation
============
* Copy the 'page_transitions' module directory in to your Drupal
sites/all/modules directory as usual.


Usage
=====
Effect to page loads is automatic, pages only slide out where the .form-submit or .page-transition class is cicked.
